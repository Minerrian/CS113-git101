Link list จะคล้ายๆ อาเรย์
Link list นั้นจะเก็บ address ขอช่องที่ติดกันอยู่ด้วย สามารถมีทั้งแบบ ทางเดียวและแบบไปกลับได้

limk list มี2แบบ
1.ทางเดียว จะไม่สามารถเรียกข้อมูลก่อนหน้านี้มาได้
2.สองทาง จะสามารถเรียกข้อมูลก่อนหน้านี้ได้

สามารถสร้างกล่องเก็บได้โดยกล่องที่เก็บข้อมูลจะถูกเรียกว่าnode ซึ่งสามารถสร้างได้โดยใช้struct
เช่น 
struct node {
     int value;
     struct node *nextnode;  
};
nextnode คือ pointer ที่ไว้ชี้ไปที่nodeถัดไป
